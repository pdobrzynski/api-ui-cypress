# contact list tests

###### How to run

```diff
## clone this repo to a local directory
git clone https://github.com/pdobrzynski/api-ui-cypress.git

## cd into the cloned repo
cd typescript-cypress

## install the node_modules
npm install

## cypress e2e tests
npm run cypress:run:e2e

## cypress api tests
npm run cypress:run:api

## cypress ui tests
npm run cypress:run:ui
```
